## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Text editor (VS Code, Sublime, etc.)
- Git installed

### Installation & Development

#### 1️⃣ Clone Repository
```bash
git clone https://github.com/gokulsaravanakkumar/portfolio.git
cd portfolio
```

#### 2️⃣ Open Locally
```bash
# Option 1: Open in browser directly
open index.html

# Option 2: Use Live Server extension in VS Code
# Install "Live Server" extension and right-click → "Open with Live Server"

# Option 3: Use Python server
python -m http.server 8000
# Then visit: http://localhost:8000
```

#### 3️⃣ View in Browser
```
http://localhost:8000
```

## 📝 Customization

### Update Personal Information
Edit `assets/portfolio.json`:
```json
{
  "name": "Your Name",
  "title": "Your Title",
  "email": "your.email@example.com",
  "phone": "+1 (XXX) XXX-XXXX",
  "location": "City, Country",
  "bio": "Your bio here",
  "social": {
    "github": "https://github.com/username",
    "linkedin": "https://linkedin.com/in/username",
    "twitter": "https://twitter.com/username"
  }
}
```

### Add New Projects
Update `projects.html` with your project details:
```html
<div class="project-card">
  <h3>Project Name</h3>
  <p>Description</p>
  <div class="tech-stack">
    <span>Tech 1</span>
    <span>Tech 2</span>
  </div>
  <a href="#">View Project</a>
</div>
```

### Modify Styles
Edit `css/styles.css` to customize:
- Colors & themes
- Fonts & typography
- Layout & spacing
- Animations & effects

## 🌐 Deployment

### Deploy to GitHub Pages

#### Step 1: Push to GitHub
```bash
git add .
git commit -m "Update portfolio"
git push origin main
```

#### Step 2: Enable GitHub Pages
1. Go to repository settings
2. Scroll to "GitHub Pages" section
3. Select branch: `main`
4. Select folder: `/ (root)`
5. Click "Save"

#### Step 3: Access Your Portfolio
- Your portfolio will be available at: `https://gokulsaravanakkumar.github.io/portfolio/`
- Changes deploy automatically on push

### Custom Domain (Optional)
1. Purchase a domain
2. In GitHub settings, add custom domain
3. Update DNS records pointing to GitHub Pages
